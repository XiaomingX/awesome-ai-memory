# Long-form Narrative & Storytelling Memory

针对长篇小说（如模拟经营、叙事文学）设计的长期记忆算法与工具，旨在维持情节连贯性与角色一致性。

## 推荐项目与算法

| 项目/算法 | 关键方法 | 适用场景 |
| :--- | :--- | :--- |
| **NovelGenerator** | 多线程叙事代理，跟踪人物视角与情感弧。 | 长篇小说全流程，情节同步。 |
| **HOMER** | 层次上下文合并（ICLR 2024），高效扩展长上下文。 | 长文本生成，避免情节遗忘。 |
| **Memobase** | 基于 Profile 的长期记忆，关注事件演化上下文。 | 人物发展、经营模拟类创作。 |
| **Awesome Story Generation**| 涵盖动态分层大纲、记忆增强等叙事论文。 | 长篇规划算法参考。 |

## 集成建议
1. **摘要管理**：先用向量库存储章节摘要，检索后再生成后续内容。
2. **人物模拟**：利用 Memobase 的 Profile 机制模拟人物的决策记忆。
3. **控制规模**：从小大纲起步，逐步扩展，防止 Token 爆炸。
